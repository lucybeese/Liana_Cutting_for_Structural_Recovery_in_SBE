# LiDAR Highlights the Crucial Role of Liana Cutting for Facilitating Structural Recovery in Tropical Forest Restoration

**NOTE:** The associated PhD thesis chapter and manuscript will be uploaded following submission of the thesis in **April 2026**.

This repository contains the workflows for **data processing, analysis, and figure generation** supporting the study:

**Beese, L. V. J. & Jucker, T. (2026).**
*LiDAR Highlights the Crucial Role of Liana Cutting for Facilitating Structural Recovery in Tropical Forest Restoration.*

The study combines **airborne LiDAR–derived structural metrics with experimental restoration treatments** to examine how **liana removal and enrichment planting influence forest structure, carbon accumulation, and canopy gap dynamics in a degraded tropical forest**.

---

# Study overview

Tropical forests store a large proportion of the Earth's terrestrial carbon but are increasingly exposed to disturbance and degradation. Restoration interventions such as **liana removal** and **enrichment planting** are commonly implemented to accelerate forest recovery, yet their structural impacts remain poorly quantified.

Using airborne LiDAR data collected across experimental restoration plots, this study evaluates how restoration treatments influence:

* **Above-ground carbon density (ACD)**
* **Canopy gap formation**
* **Canopy gap closure**

By linking structural metrics derived from LiDAR with experimental restoration treatments and topographic variables, the study provides insights into the mechanisms through which restoration interventions influence forest recovery trajectories.

        ↓

The workflow allows reproducible evaluation of how restoration treatments influence structural recovery in tropical forests.

# Code overview

The `code/` directory contains scripts used to:

* Process airborne LiDAR data and derive **forest structural metrics**
* Calculate **above-ground carbon density (ACD)**
* Quantify **canopy gap formation and gap closure**
* Integrate LiDAR-derived metrics with **experimental restoration treatments and topographic variables**
* Fit statistical models evaluating treatment and environmental effects on forest recovery
* Generate figures and summary tables used in the manuscript and supplementary information

Scripts are organised sequentially to reflect the analytical workflow.

# Data dictionary (key variables)

The following variables are central to the analyses:

| Variable                 | Description                                                                                    |
| ------------------------ | ---------------------------------------------------------------------------------------------- |
| **ACD**                  | Above-ground carbon density derived from LiDAR structural metrics                              |
| **New gap volume**       | Volume of newly formed canopy gaps per hectare per year                                        |
| **Existing gap closure** | Proportion of existing canopy gaps closing per year                                            |
| **Treatment**            | Restoration treatment applied to plots (control, enrichment planting, liana removal)           |
| **HALP**                 | Height above the lowest point; a topographic index representing relative hydrological position |

# How to run the analysis

1. Clone the repository
2. Install required R packages
3. Populate the `data/` directory following the instructions provided
4. Run scripts in the `code/` directory in the order indicated within the scripts
5. Outputs (figures) will be written to the `figures/` directory where instruction on changing working directory are followed
